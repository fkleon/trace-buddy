Matrix containing the output colors at every pixel of the final image set by the renderer. The color information are stored as RGB vectors, with each component in the interval 0..1.

Constructors

OutputMatrix(int rows, int columns)

Initializes a output matrix of size rows*columns and initially set all color information to black.
