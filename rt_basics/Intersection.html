Intersection class

The Intersection class represents a ray/object intersection. It contains the distance to the hit point, the affected Primitive, and the hit point as Point3D.

If there is no hit, distance will be negative.

Constructors

new Intersection([num distance = -1, Primitive prim = null, Point3D hitPoint]) #

Creates a new Intersection object.

Default values are distance = -1, prim = null.
